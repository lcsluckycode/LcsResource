SpringBoot快速入门

SpringBoot 快速入门

springboot是spring家族中微型框架，其设计目的是用来简化新Spring应用的初始搭建以及开发过程。 Spring Boot可以轻松创建独立的，生产级的基于Spring的应用程序。

使用SpringBoot可以：

  • 简单、快速、方便地搭建Web项目；
  • 对主流开发框架的无配置集成；
  • 极大提高了开发、部署效率。

环境准备

环境准备:

1.Jdk8.0以上。

2.Maven3.x以上，需要设置jdk8的运行环境。

3.开发工具：sts、idea。

快速启动一个springboot项目：

快速入门

步骤如下：

  1. 在 https://start.spring.io/ 上获取项目
  2. 导入项目
  3. 编写路由
  4. 运行程序

获取SpringBoot项目

打开网站： https://start.spring.io/ 获取一个Demo项目。

导入项目

在IDEA下选择 file -> open 即可。

模块说明

SpringbootApplication： 一个带有 main() 方法的类，用于启动应用程序

SpringbootApplicationTests：一个空的 Junit 测试类，它加载了一个使用 Spring Boot 字典配置功能的 Spring 应用程序上下文

application.properties：一个空的 properties 文件，可以根据需要添加配置属性

pom.xml： Maven 构建说明文件

编写程序

在com.example.demo包下新建一个HelloController并编写代码：

package com.example.demo.controller;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @RequestMapping("/hello")
    public String hello(){
        return "hello world";
    }

}
