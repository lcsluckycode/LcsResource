Phoenix&Spark

Phoenix 与 Spark

Phoenix-Spark 插件扩展了 Phoenix 的 MapReduce 支持，允许 Spark 将 Phoenix 表加载为 RDD 或 DataFrame ，并使它们能够持久地返回 Phoenix。

Phoenix-Spark 也在 Apache Maven Repository 托管。您可以将以下内容添加到 pom 中：

使用 Phoenix 和 Spark 相结合的目的是将phoenix做存储，spark做计算层。这样就结合了phoenix查询速度快和spark计算速度快的优点。

这种方案运维成本不高，只需关注HDFS即可

所以 Phoenix 和 Spark 结合简直完美！

要将他们结合可以导入 pom 配置。

    <properties>
        <java.version>1.8</java.version>
        <spark.version>2.4.0</spark.version>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>

    <dependencies>
    <dependency>
        <groupId>org.apache.hadoop</groupId>
        <artifactId>hadoop-client</artifactId>
        <version>2.7.7</version>
    </dependency>
    <dependency>
        <groupId>org.apache.phoenix</groupId>
        <artifactId>phoenix-core</artifactId>
        <version>5.0.0-HBase-2.0</version>
    </dependency>
    <dependency>
        <groupId>org.apache.phoenix</groupId>
        <artifactId>phoenix-spark</artifactId>
        <version>5.0.0-HBase-2.0</version>
    </dependency>
    <dependency>
        <groupId>org.apache.spark</groupId>
        <artifactId>spark-core_2.11</artifactId>
        <version>${spark.version}</version>
    </dependency>
    <dependency>
        <groupId>org.apache.spark</groupId>
        <artifactId>spark-sql_2.11</artifactId>
        <version>${spark.version}</version>
    </dependency>
</dependencies>

将 Phoenix 表加载为 DataFrame

通过DataSource API可以把 Phoenix 表加载成 DataFrame，只需要给定一个表名和 HBase 的 Zookeeper 地址。

// 创建本地的SparkSession 对象
SparkSession spark = SparkSession.builder().appName("BKHBase")
.master("local").getOrCreate();

// 创建DataSet
Dataset<Row> load = spark.read().option("table", "T_HOTEL")
.option("zkUrl", "127.0.0.1:2181")
.format("org.apache.phoenix.spark").load();

计算

表名：T_BKYJ_N_TZ

字段名称 类型 介绍
ID VARCHAR 主键ID
BKID VARCHAR 布控ID（主表ID）
TZLX VARCHAR 布控特征类型（1：姓名；2：身份证；3：手机号码；4：手机IMEI码；5：手机IMSI码；6：QQ号码；7：微信号码；8：车牌种类，车牌号码；9：性别）
DXLX VARCHAR 布控对象类型（1：人员；2：车辆；）
TZXX VARCHAR 特征信息

表名：T_BKYJ_N_JBXX

字段名称 类型 介绍
ID VARCHAR 主键ID
BKMC VARCHAR 布控名称 *
SJFW VARCHAR 数据范围（1：全市；2：全省；3：全国）
KSSJ VARCHAR 开始时间
JSSJ VARCHAR 结束时间

表名：dw.znb_22_113200003

字段名称 类型 介绍
XM VARCHAR 姓名
GMSFHM VARCHAR 公民身份号码

通过布控特征表T_BKYJ_N_TZ,布控详细信息表T_BKYJ_N_JBXX，人员基本信息表dw.znb_22_113200003关联得到布控人员的信息。

  • 最后展示的列及其列名为：布控id（bkid）、布控名称（bkmc）、数据范围（sjfw）、开始时间（kssj）、结束时间（jssj）、姓名（name）。

  • 表T_BKYJ_N_TZ的特征信息由特征类型决定。

  • 若特征信息为公民身份号码，与表dw.znb_22_113200003关联得到该布控人员姓名。

  • 若特征信息既不是姓名也不是公民身份号码，则过滤该数据。

  • 表T_BKYJ_N_JBXX 将其数据范围改为对应的范围（1 -> 全市2 -> 全省 3 -> 全国 其他情况为未知）。

处理前两张表

// 定义一个本地的Spark
SparkSession spark = SparkSession.builder().appName("BKHBase").master("local").getOrCreate();

// 加载HBase表数据
Dataset<Row> a = spark.read().option("table", "T_BKYJ_N_TZ")
        .option("zkUrl", "116.196.80.141:2181")
        .format("org.apache.phoenix.spark").load();
Dataset<Row> b = spark.read().option("table", "T_BKYJ_N_JBXX")
        .option("zkUrl", "116.196.80.141:2181")
        .format("org.apache.phoenix.spark").load();
Dataset<Row> c = spark.read().option("table", "dw.znb_22_113200003")
        .option("zkUrl", "116.196.80.141:2181")
        .format("org.apache.phoenix.spark").load();

a.createOrReplaceTempView("bktz");

b.createOrReplaceTempView("jbxx");

c.createOrReplaceTempView("gmxx");


// 查询 特征为 1 / 2 的数据
spark.sql("SELECT BKID,TZLX,TZXX from bktz where TZLX = '1' or TZLX = '2'")
        .createOrReplaceTempView("t1");

// t1、jbxx 两表关联后获取相关信息，同时将sjfw（数据范围）这一列数据替换为相应的名称
spark.sql("select t1.BKID bkid,t1.TZXX tzxx,jbxx.BKMC bkmc," +
        "(case when jbxx.SJFW ='1' then '全市' when jbxx.SJFW ='2' then '全省' when jbxx.SJFW ='3' then '全国' else '未知' end)sjfw" +
        ",jbxx.KSSJ kssj,jbxx.JSSJ jssj from t1 left join jbxx on t1.bkid = jbxx.id")
        .createOrReplaceTempView("t2");
// t2 和 gmxx 关联，将身份证号码转换成对应的姓名
spark.sql("select t2.bkid," +
        "(case when t2.tzxx = t3.GMSFHM then t3.xm else t2.tzxx end)tzxx" +
        ",t2.bkmc,t2.sjfw,t2.kssj,t2.jssj from t2 left join (select GMSFHM,xm from gmxx)t3 on t2.tzxx=t3.GMSFHM").show();

将 csv 数据导入至HBase

建表

create table dw.znb_03_143200081(XM varchar, XB varchar, CSRQ varchar, MZ varchar, LGMC varchar, LGDZQH varchar, SSQBM varchar, XZ varchar, LGDZ varchar, ZJLX varchar, RZSJ varchar, TFSJ varchar, RZFH varchar, ZJHM varchar, XP varchar, LKBH varchar, LGBM varchar CONSTRAINT PK PRIMARY KEY (RZSJ,ZJHM));

导入数据：

SparkSession spark = SparkSession.builder()
                .appName("csvToHBase")
                .master("local").getOrCreate();
// 加载csv数据集
Dataset<Row> file = spark.read()
        .option("header", "true")
        .csv("./src/main/resources/cc.csv");
// 获取csv数据集的所有列名
String[] columns = file.columns();
// 将存放csv数据集所有列名的数组转化成大写之后放入list
ArrayList<String> fileList = new ArrayList<String>();
for (int i = 0; i < columns.length; i++) {
        fileList.add(columns[i].toUpperCase());
}
System.out.println(fileList.toString());
// 读取HBase表T_BKYJ_N_JBXX的所有列名
String[] hbaseColumn = spark.read()
        .option("table", "dw.znb_03_143200081")
        .option("zkUrl", "116.196.80.141:2181")
        .format("org.apache.phoenix.spark").load().columns();
// 将数组转换成集合
List<String> hbaseList = Arrays.asList(hbaseColumn);
// 获取两个集合之间的交集
fileList.retainAll(hbaseList);
System.out.println(fileList);
// select a,b,c from table
// 将查询的列名进行字符串拼接
String c = fileList.toString().substring(1, fileList.toString().length() - 1);
System.out.println(c);
// 注册一个临时表
file.createOrReplaceTempView("t");
// 从 csv 数据集中加载共有列的数据，将其存入HBase表T_BKYJ_N_JBXX中
spark.sql("select " + c + " from t")
        .write().mode(SaveMode.Overwrite)
        .option("table", "dw.znb_03_143200081")
        .option("zkUrl", "116.196.80.141:2181")
        .format("org.apache.phoenix.spark").save();
spark.stop();
