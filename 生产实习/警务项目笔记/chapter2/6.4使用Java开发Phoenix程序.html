使用编程语言开发Phoenix程序

4 使用开发语言编写 Phoenix 程序

在实际的项目开发中，我们还是要使用代码去操作数据库的，接下来我们学习如何以 JDBC 连接 Phoenix 操作 HBase。

4.1 使用 JDBC 连接 Phoenix

Phoenix 提供了完全适配 JDBC 的 API，连接 Phoenix 可以像连接关系型数据库一样来使用 JDBC 来操作Phoenix，不过值得我们注意的是，提交 SQL 时需要注意使用 Phoenix 支持的 SQL，Phoenix 的 SQL 和其他关系型数据库还是有一些不同之处的。

接下来我们通过一个例子来查看如何使用 Java 连接 Phoenix，编写代码之前需要导入 phoenix-core-5.0.0-HBase-2.0.jar 包至你的项目中，如果你使用的是 maven 项目可以在pom 文件中加入依赖如下：

<dependencies>
    <dependency>
        <groupId>org.apache.hadoop</groupId>
        <artifactId>hadoop-client</artifactId>
        <version>2.7.7</version>
    </dependency>
    <dependency>
        <groupId>org.apache.phoenix</groupId>
        <artifactId>phoenix-core</artifactId>
        <version>5.0.0-HBase-2.0</version>
    </dependency>
</dependencies>

JDBC 操作和普通的关系型数据库一样：

  1. 注册驱动
  2. 获取连接对象
  3. 执行SQL语句
  4. 提交

接下来就可以开始编写Java程序了，开发 Phoenix 程序和普通的 JDBC 应用没啥区别，来看个例子：


import com.sun.org.apache.xml.internal.utils.StringComparable;
import org.apache.hadoop.hbase.CompareOperator;
import org.apache.hadoop.hbase.filter.*;
import org.apache.hadoop.hbase.util.Bytes;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.sql.Statement;

public class HBaseJavaAPI {
    static {
        // 1. 注册驱动
        try {
            Class.forName("org.apache.phoenix.jdbc.PhoenixDriver");
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        }
    }

    /*
     * 连接数据库
     */
    public static Connection getConnection(){

        Connection conn = null;
        try {
            //2, 获取连接对象
            conn = DriverManager.getConnection("jdbc:phoenix:116.196.80.141:2181");
        } catch (SQLException e) {
            e.printStackTrace();
        }
        return conn;
    }


    /**
     * 创建表
     */
    public static void createTable(String sql){
        Connection connection = getConnection();
        try {
            Statement statement = connection.createStatement();
            //3. 执行SQL语句
            statement.execute(sql);
            //3. 提交
            connection.commit();
            connection.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    /**
     * 添加或者更新数据
     */
    public static void upsert(String sql){
        Connection connection = getConnection();
        try {
            Statement statement = connection.createStatement();
            statement.executeUpdate(sql);
            connection.commit();
            connection.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    public static void main(String[] args) {
        String createSQL = "create table student(ID INTEGER PRIMARY KEY,INFO.NAME VARCHAR,INFO.AGE VARCHAR,INFO.SEX VARCHAR)";
        String upsertSQL = "upsert into student values(1,'zhangsan','20','male')";
        createTable(createSQL);
        upsert(upsertSQL);
    }
}

执行上述代码后，在命令行执行查询：

0: jdbc:phoenix:127.0.0.1:2181> select * from student;
+-----+-----------+------+-------+
| ID  |   NAME    | AGE  |  SEX  |
+-----+-----------+------+-------+
| 1   | zhangsan  | 20   | male  |
+-----+-----------+------+-------+

可以发现刚刚编写的 Java 代码完成了创建表与插入数据的工作。

4.2 使用 Python 开发 Phoenix 程序

安装驱动，Phoenix目前没有部署驱动程序，因此用户必须首先构建驱动程序并自行安装：

cd /app/phoenix5.0/python/
pip install -r requirements.txt 
python setup.py install

启动服务：切换目录在 Phoenix 的 bin 目录下执行 queryserver.py 文件：

cd /app/phoenix5.0/bin/
python2 ./queryserver.py

查看 8765 端口是否正常：

netstat -npl |grep 8765
tcp6       0      0 :::8765                 :::*                    LISTEN      24825/java

编写 Python 文件：

import phoenixdb
import phoenixdb.cursor

database_url ='http://localhost:8765/'
conn = phoenixdb.connect(database_url,autocommit = True)

cursor = conn.cursor()
cursor.execute("CREATE TABLE users(id INTEGER PRIMARY KEY,username VARCHAR)")
cursor.execute("UPSERT INTO user VALUES(?，?)"，(1，'admin'))
cursor.execute("SELECT * FROM users")
print(cursor.fetchall())

cursor = conn.cursor(cursor_factory = phoenixdb.cursor.DictCursor)
cursor.execute("SELECT * FROM users WHERE id = 1")
print(cursor.fetchone()['USERNAME'])

执行即可查看到如下结果：

[[1, 'admin']]
admin
