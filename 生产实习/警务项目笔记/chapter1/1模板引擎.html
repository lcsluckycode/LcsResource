模板引擎快速入门

怎么给服务器的用户呈现一个漂亮的页面呢？

肯定需要用到 html、css ，如果想要更炫的效果还要加入 js ，问题来了，这么一大堆字段串全都写到视图中通过 return 返回，这样定义就太麻烦了吧，因为定义字符串是不会出任何效果和错误的，如果有一个专门定义前端页面的地方就好了。

为此我们可以使用模板。

创建模板

首先在项目的 pom.xml 文件中加入模板引擎的依赖库：

在 <dependencies> 标签中加入如下配置：

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>

然后在 resources 目录下创建一个 templates 模板文件夹。

使用模板

接下来在 templates 目录下创建一个名为 index.html 的网页文件，在该文件中写入如下内容：

<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
</head>
<body>
    <h1 th:text="|欢迎你, ${user.name}!|"></h1>
</body>
</html>

关键代码为： <span th:text="|欢迎你, ${user.name}!|">

其中 ${user.name} 为获取user对象的name属性。

编写好网页文件之后，在Java项目中创建一个TmplateController。

写入如下代码：

package com.example.demo.controller;

import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.RequestMapping;

import java.util.HashMap;
import java.util.Map;

@Controller
public class TmplateController {

    @RequestMapping("/index")
    public String demo(Model model) {
        Map<String,String> user = new HashMap<>();
        user.put("name","张三丰");
        model.addAttribute("user", user);
        return "index";
    }
}
